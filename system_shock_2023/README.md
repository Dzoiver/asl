# System Shock Remake
Auto Splitter and Load Remover for System Shock (Remake).
## Usage
* Download the script.
* For LiveSplit add Edit Layout... > Control > Scriptable Auto Splitter to your layout.
* Open the layout settings, and point it to the file you downloaded.
